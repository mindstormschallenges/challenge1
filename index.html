PiStorms Challenge

Introduction

PiStorms Challenge is the first of challenge created by Lego Mindstorms community for developers worldwide. The idea is to promote innovation from the own community. In this challenge the idea is to innovate using as product base the PiStorms case:

and the challenge is to modify/change or redesign this case in order to create better robots using this case in combination with Raspberry Pi & Raspberry Pi 2 with the hat PiStorms.

The base elements used to begin the challenge are:

  • http://www.philohome.com/pistorms/pistorms.htm
  • http://www.mindsensors.com/rpi-cases-and-mounts/156-pistorms-frame-cad-files

How to participate in the challenge?

Participate in the challenge is very easy, fork the Github repository: http://github.com/mindstormsChallenges/challengeCase . In this repo, you will see the materials to work. All participants has the opportunity to make pull request with this innovation for the case from 15/3/2016 00:00am to 15/5/2016 00:00am every contribution to be valid has to include:

  • folder with the github user
  • a cad/ldraw with the model
  • a image with the model

The panel of experts will valorate:

  • the creativity
  • the combination with lego construction kit
  • the ease to put PiStorms hat
  • the ease to put PiStorm battery set
  • the easy to put sensor & motors

The prize

The best design will receive a 3 case designers will receive a PiStorms unit to use the case modeled with PiStorms. Mindsensors, the main sponsor of this challenge will deliver the unit to your home worldwide.
