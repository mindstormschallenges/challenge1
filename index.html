PiStorms Challenge

PiStorms Challenge is the first of challenge created by Lego Mindstorms community for developers worldwide. The idea is to promote innovation from the own community. In this challenge the idea is to innovate using as product base the PiStorms case

The challenge invites 3D Designers to modify/change or redesign this case in order to create better robots using this case in combination with Raspberry Pi & Raspberry Pi 2 with the board PiStorms.

Read the rules to participate in the challenge to get a PiStorms board.

Rules to participate in the Challenge

The base elements used to begin the challenge are:

  • http://www.philohome.com/pistorms/pistorms.htm
  • http://www.mindsensors.com/rpi-cases-and-mounts/156-pistorms-frame-cad-files

Participate in the challenge is very easy, fork the Github repository: https://github.com/mindstormschallenges/challenge1. In this repo, you will see the materials to work. All participants has the opportunity to make pull request with this innovation for the case from 15/3/2016 00:00am to 15/5/2016 00:00am every contribution to be valid has to include:

  • folder with your github user
  • a cad/ldraw with the model
  • a image with the model

Criteria to evaluate

The panel of experts will valorate:

CREATIVITY

We valorate the creativity of the case and the possible applications to build simple and complex robots.

EASE TO COMBINE

The ease to combine with the construction system is a critical point.

EASE TO PUT PISTORMS & BATTERY

The ease to put a Raspberry Pi with the board PiStorms is important. The robot builder need to extract the battery in a easy way.

EASE TO PUT SENSORS & MOTORS

Think in some creative ways to put sensors &s motors in a easy way.

The prize

The best 3 desigsn will receive a PiStorms unit to use the case modeled with PiStorms. Mindsensors, the main sponsor of this challenge will deliver the unit to your home worldwide.
